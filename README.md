# Vue Boilerplate SPA

## Project setup

Link Demo: [http://vue-spa-kit.surge.sh/](http://vue-spa-kit.surge.sh/)

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
